## Project Summary
CyberAware Dashboard is a no-login cybersecurity training portal designed for community organizations. 
It includes short training modules, phishing simulations, incident-response scenarios, and a points-based dashboard.
An admin dashboard provides anonymized, aggregate progress analytics for an organization cohort.

## Tech Stack (Planned)
- Frontend: React (JavaScript)
- Backend: Python FastAPI
- Database: SQLite (prototype), optional PostgreSQL later
- No-login approach: anonymous participant ID stored locally; optional Org Code + Admin Code
